The console is the front-end of a 3 tier application:

  • A database storing end-user information.
  • A Web tier which control database accesses and updates.
  • This lightweight console running in end-users' Web browsers.

The console is mainly built with the Dojo Toolkit [1]. Dojo is an open source library supported by major companies [2]. It is available under two licenses [3]: BSD license and Academic Free License.

Some icons have been produced by Mark James [4], freely available to anyone under the Common Creative Attribution 2.5 license [5].

The middle-tier and the database are operated with Google App Engine (GAE) infrastructure [6]. Thanks to GAE architecture, this application should run from anywhere efficiently, following the cloud computing paradigm.
